# 个人健康管理系统 java19

## 项目概述

个人健康管理系统是一款基于前后端分离架构设计的应用，旨在帮助用户更好地管理个人健康数据，通过健康食谱、运动及饮食信息的记录与分析，为用户的健康管理提供支持。

## 技术栈

- 后端：Java、Springboot、MyBatis、MySQL
- 前端：jQuery、HTML、Vue

## 功能模块

### 用户模块

- 首页：提供健康食谱、健康数据、饮食分析、运动分析等概览信息
- 健康数据管理：管理用户的健康数据
- 饮食信息管理：
  - 饮食分析：对用户饮食数据进行分析
  - 饮食记录：记录用户饮食信息
- 运动信息管理：
  - 运动分析：对用户运动数据进行分析
  - 运动计划：制定用户运动计划
  - 运动记录：记录用户运动数据

### 管理模块

- 健康食谱管理：管理健康食谱信息
- 管理员管理：对管理员账户进行管理

## 系统角色

- 普通用户：使用系统进行健康数据管理
- 管理员：负责健康食谱和管理员账户的管理

## 运行环境

- 后端：Java 1.8+、MySQL 5.7+
- 前端：现代浏览器支持

- 基于前后端分离的设计，易于扩展和维护
